# Security Policy

## Supported Versions

This project follows [semantic versioning](https://semver.org/) (`MAJOR.MINOR.PATCH`).

I only provide security patches for the most recently released major version line — that is, versions whose `MAJOR` number matches the highest `MAJOR` currently published for this project. All earlier major versions are end-of-life and receive no further security updates.

| Version range                          | Security support status |
| -------------------------------------- | ------------------------ |
| Highest MAJOR line (e.g. `N.x.y`)      | Supported                |
| Any version with `MAJOR` < `N`         | Unsupported / EOL        |

## Reporting a Vulnerability

Please report security vulnerabilities via the GitHub Security Advisories "Report a vulnerability" feature for this repository. Do not file public GitHub issues for security vulnerabilities.
